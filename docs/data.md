# Data Access

Use this page to document the datasets, data products, and access notes your group actually used.

Start with the [ESIIL Data Library](https://cu-esiil.github.io/data-library/innovation-summit-2025/) and replace this page with your group’s specific sources.

For each dataset, record:

- dataset name
- link or storage location
- what it contains
- why it matters for your question
- any license, access, or quality notes
